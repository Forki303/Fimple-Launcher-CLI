## Русский

**Fimple Launcher CLI** — это легковесная и быстрая консольная версия лаунчера Fimple для Minecraft. Идеально подходит для тех, кто предпочитает минимализм, управление через терминал или играет на слабых ПК.

### Основные возможности
* **Управление через консоль:** Быстрый запуск игры без тяжелого графического интерфейса.
* **Выбор версий:** Поддержка и запуск любых версий Minecraft.
* **Гибкие настройки:** Удобная конфигурация параметров запуска прямо из терминала.

### Требования
* **Java 21+** — обязательна для запуска актуальных и новых версий Minecraft (начиная с 1.17+).
    * [Скачать JDK 21 с официального сайта Oracle](https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html)
* Установленный **Git** (для клонирования репозитория).

### Установка и запуск

Выполните следующие команды в вашем терминале:

```bash
# Клонируйте репозиторий проекта
git clone [https://github.com/Forki303/Fimple-Launcher-CLI.git](https://github.com/Forki303/Fimple-Launcher-CLI.git)

# Перейдите в папку с проектом
cd Fimple-Launcher-CLI

```
python main.py

## English

**Fimple Launcher CLI** — a lightweight and fast command-line version of the Fimple Launcher for Minecraft. Perfect for those who prefer minimalism, terminal-based control, or playing on low-end PCs.

### Key Features

* **CLI Control:** Quick game launch without a heavy graphical user interface.
* **Version Selection:** Support and launching for any Minecraft version.
* **Flexible Settings:** Easy configuration of launch parameters right from your terminal.

### Requirements

* **Java 21+** — required to run modern and new versions of Minecraft (1.17+).
* [Download JDK 21 from the official Oracle website](https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html)


* **Git** installed on your system.

### Installation & Setup

Run the following commands in your terminal:

```bash
# Clone the repository
git clone [https://github.com/Forki303/Fimple-Launcher-CLI.git](https://github.com/Forki303/Fimple-Launcher-CLI.git)

# Navigate to the project directory
cd Fimple-Launcher-CLI

```

python main.py
